### **üöó Ride Management**
- **Create Rides**: Post your ride with departure time, destination, available seats, and fare per person

- **Real-time Tracking**: Live GPS tracking for enhanced safety during rides
- **Fare Splitting**: Automated cost calculation and transparent fare distribution
- **Ride History**: Track all your past rides as driver or passenger

### **üí¨ Communication & Social Features**
- **In-Ride Messaging**: Chat directly with ride participants once you join a ride
- **Group Chat**: Communicate with all passengers and driver in a dedicated chat room
- **Real-time Updates**: Instant notifications for ride status changes and messages
- **Contact Sharing**: Secure exchange of contact information between ride participants
- **Emergency Features**: Quick access to emergency contacts and campus security

### **üîç Smart Ride Discovery**

## üîß **Installation & Setup**

### **Prerequisites**
```bash
Flutter SDK 3.7.0+
Dart 3.0.0+
Android Studio / VS Code
Firebase CLI
```

### **Quick Start**
```bash
# Clone the repository
git clone https://github.com/0Jahid/RideZO_App-main.git

# Navigate to project directory
cd RideZO_App-main

# Install dependencies
flutter pub get

# Run the application
flutter run
```

### **Firebase Configuration**
1. Create a new Firebase project
2. Enable Authentication, Firestore, and Cloud Messaging
3. Download `google-services.json` (Android) and `GoogleService-Info.plist` (iOS)
4. Place configuration files in respective platform directories
